🚍 Projeto ExpressoEDA – Simulação de Sistema de Autocarros 🚌
Este projeto implementa uma simulação de uma empresa de transporte de passageiros ponto a ponto, focando a gestão de autocarros, paragens e passageiros. Desenvolvido em C++, o sistema utiliza estruturas de dados como listas ligadas e árvores binárias para modelar a operação do serviço de transporte.

🎯 Objetivo
Simular o funcionamento de uma rede de transporte de autocarros, incluindo criação e remoção de autocarros, gestão de filas de espera de passageiros nas paragens, movimentação dos autocarros entre paragens, e atualização das informações dos passageiros.

⚙️ Funcionalidades
🚏 Criação de autocarros com capacidade definida e motoristas;
👥 Gestão de passageiros, incluindo criação e remoção com base em probabilidades;
📋 Estruturas de dados para representar filas de espera e paragens;
🔄 Movimento automático dos autocarros entre as paragens;
📝 Atualização e registo do estado dos autocarros e passageiros;
👨‍💼 Funcionalidades administrativas como alteração de motoristas e remoção de bilhetes.
🏗️ Estrutura do Projeto
main.cpp: programa principal que controla o fluxo da simulação;
autocarro.cpp / .h: funções e estruturas para autocarros;
passageiro.cpp / .h: manipulação de passageiros e respetivas probabilidades;
paragem.cpp / .h: gestão das paragens e filas;
🛠️ Outras funções auxiliares para a gestão da rede.
⚠️ Nota
🚧 A opção de paragem intermédia não está implementada.
🚀 Possíveis Expansões Futuras
📁 Implementação da leitura e escrita de dados em ficheiros;
🎨 Interface gráfica para visualização do sistema;
⚡ Otimizações nas estruturas de dados;
🚌 Novos tipos de autocarros e rotas dinâmicas.
