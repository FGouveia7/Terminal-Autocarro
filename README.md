# Project — Restaurant Simulation (CLI)

This repository contains a **text-based (CLI) simulation** written in **C++**, where the program maintains and prints the state of a restaurant: **tables**, **entry queue**, and **counter/bar**. Interaction is done through terminal menus (e.g., “next” / “options”).

The entry point is `Main/1main.cpp` (the `main` function), which initializes the core structures and enters the menu loop (`menu_s` / `menu_o`).

## Features

- **Restaurant state**: prints the current state (tables, queue, counter).
- **Tables**: dynamic creation/initialization of a variable number of tables.
- **Customers and queue**: customer generation and entry-queue management.
- **Counter/bar**: initialization and management of the counter (customers/specialty/cycles).
- **Interaction menus**: advance cycles (“Next”) and execute actions (“Options”).

## Tech stack

- **C++20** (console application)
- **CMake** (build)

## Requirements

- **CMake 3.30+** (per `Main/CMakeLists.txt`)
- **A C++20-capable compiler** (e.g., MSVC / MinGW / clang)

To verify:

```bash
cmake --version
```

```bash
cd <cloned-folder>
```

## Usage (build and run)

The `CMakeLists.txt` lives under `Main/`, so the commands below point to that directory.

### Windows (PowerShell/CMD)

```bash
cmake -S .\Main -B .\build
cmake --build .\build --config Debug
.\build\Debug\Projeto.exe
```

If your generator does not use per-config folders (e.g., Ninja/MinGW), the executable may be located at:

```bash
.\build\Projeto.exe
```

## Program “commands” (in the terminal)

When the program starts, it prints the initial state and asks you to choose an option:

- **`s`**: Next (advances the simulation via `menu_s`)
- **`o`**: Options (opens operations via `menu_o`)

## Project structure

- `Main/CMakeLists.txt`: CMake configuration (builds the `Projeto` executable)
- `Main/1main.cpp`: entry point (main loop + menus)
- `Main/2estruturas.h`: core structs (`cliente`, `Mesa`, `Balcao`)
- `Main/3cliente.*`: customer logic
- `Main/4fila.*`: queue logic
- `Main/5mesa.*`: table logic
- `Main/6balcao.*`: counter logic
- `Main/7Menus.*`: menus/actions
- `Main/8carregarEguardar.*`: load/save (persistence)

## Common issues

- **CMake can’t find a compiler**: on Windows, install “Desktop development with C++” (Visual Studio) or configure MinGW/clang, then rerun `cmake -S ... -B ...`.
- **Build folders showing in git status**: directories like `cmake-build-debug/` and `build/` are generated by CMake/your IDE and generally **should not be committed**.
